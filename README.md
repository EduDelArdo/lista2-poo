# Lista de Exercícios II - Parte 1 (POO)

Este repositório contém as soluções da Lista de Exercícios II de Programação Orientada a Objetos, desenvolvida em Python. O projeto foi estruturado de forma inteiramente modular, separando Modelos (Models), Repositórios (Repository) e Interfaces de Usuário (UI).

## Estrutura do Repositório

O projeto está dividido em cinco módulos principais, cada um contendo o seu código-fonte e um arquivo de texto com as respostas conceituais das perguntas de fixação exigidas nos slides do professor:

* **`questao_01/`**: Sistema de Mídias Educacionais (Contrato formal por Herança - Classes Abstratas / ABC)
* **`questao_02/`**: Folha de Pagamento de Funcionários (Polimorfismo e Sobrescrita de Métodos com ABC)
* **`questao_03/`**: Sistema de Notificações (Contrato formal por Herança - ABC)
* **`questao_04/`**: Sistema de Impressão de Documentos (Tipagem Estrutural - Protocol / Duck Typing)
* **`questao_05/`**: Sistema de Armazenamento Dinâmico (Comparativo prático entre ABC vs Protocol)

## Como Executar as Questões

Como configurámos o sistema com importações diretas para evitar conflitos de caminhos no terminal, para executar qualquer uma das questões basta **entrar na pasta correspondente** pelo terminal e rodar o arquivo `main.py`.

Abra o terminal do VS Code na raiz do projeto (`LISTA2-POO`) e utilize os comandos abaixo para testar cada sistema:

### Questão 1 - Mídias Educacionais
```bash
cd questao_01
python main.py
cd ..
```

### Questão 2 - Sistema de Funcionários
```bash
cd questao_02
python main.py
cd ..
```

### Questão 3 - Sistema de Notificações
```bash
cd questao_03
python main.py
cd ..
```

### Questão 4 - Sistema de Impressão
```bash
cd questao_04
python main.py
cd ..
```

### Questão 5 - Sistema de Armazenamento
```bash
cd questao_05
python main.py
cd ..
```
